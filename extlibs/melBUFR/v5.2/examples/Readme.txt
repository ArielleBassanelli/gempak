This tar file contains examples using the MEL BUFR Library V2.0 and latter 
versions.  There are 16 examples and some have several cases.  The example are
used to illkustrate some base BUFR concepts and how to use the MEL BUFR
Library.  Documentation of the examples is included as a MS Word file
and as a PostScript file.

Case C of Example 16, requires the downloading of an additional file, s950108l.
This file is ~5792230 bytes long comnpressed and 27+Mb uncompressed. 
This file contains a worldwide set of surface observations and is used
to perform timeing tests.  Case's A nd B use a shortend version of this file and 
is provided in the initial tar file. 

Additional examples will be added as needed or the become available.
